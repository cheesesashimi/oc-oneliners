# oc-oneliners

This repo contains various Python and shell scripts I use throughout the course
of my daily work as an OpenShift developer. They're likely not very useful for
non-OpenShift developers.

Being that this is more of a personal repository, it has accumulated a lot of
cruft which I then need to clean up. However, that is the beauty of a version
control system: If the script was useful in the past but is no longer useful,
one can just delete it with the full knowledge that the script itself still
exists within the repo history.

With that said, if you've found a dead link to a script that I've posted
before, you may want to check the repo history for the script since it was
likely deleted or moved.
